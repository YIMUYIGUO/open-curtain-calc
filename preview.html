🌟 项目介绍

OpenCurtainCalc 是一个功能全面的开源幕墙计算系统，旨在替代商业软件如豪沃克幕墙计算软件。系统提供完整的幕墙设计、计算、分析和报告生成功能。

🚀 查看GitHub仓库 🎮 查看演示 ⚡ 快速安装

🎮 核心特性

🏗️

结构计算

立柱、横梁、连接件等结构计算，支持多种材料类型和荷载组合。

💨

荷载分析

风荷载、地震荷载、温度荷载等复杂荷载分析，符合国标规范。

📊

材料统计

自动材料用量计算和成本估算，支持Excel报告导出。

🔗

节点设计

连接节点设计和验算，可视化节点构造和受力分析。

📄

计算书生成

自动生成符合国标的计算书，支持Word/PDF格式导出。

🤖

AI辅助设计

智能优化建议和异常检测，提高设计效率和质量。

⚡ 快速启动

# 克隆项目
git clone https://github.com/YIMUYIGUO/open-curtain-calc.git
cd open-curtain-calc

# 启动开发环境 (需要Docker)
./scripts/deploy.sh dev up

# 访问应用
前端界面: http://localhost:3000
后端API: http://localhost:8000
API文档: http://localhost:8000/docs
